# 🚀 GitHub Profile Analyzer (Python + GenAI)

## 📌 Overview
This project is a **GitHub Profile Analyzer** built using Python that automates the evaluation of a developer’s GitHub profile.

It fetches repository data using GitHub API, analyzes key metrics, and generates **AI-powered professional summaries**.

---

## 🎯 Features
- 🔍 Fetch GitHub repositories using API  
- 📊 Analyze stars, forks, and languages  
- 📈 Visualize most-used languages  
- 🤖 Generate AI-based professional insights  
- 📄 Export report for recruiters  

---

## 🛠️ Tech Stack
- Python  
- Requests (API handling)  
- Pandas (data analysis)  
- Matplotlib (visualization)  
- OpenAI / GenAI
